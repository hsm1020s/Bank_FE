# 페이즈 5 — 직원 확장 / 운영

## 1. 페이즈 목표

직원 채널의 **나머지 모든 기능**을 채운다. 창구거래 / 정정·취소 / 약정 관리 / 실행 / 부실 / 압류 / 사고대응 / 시스템·배치 / 규제보고 / 결재 / 상품마스터 — **운영 가능한 은행** 완성.

## 3. 페이즈 5 완료 정의 (DoD)

- [ ] EMP-001 직원망 IP만 + PKI + OTP, 신규 IP/MAC 시도 즉시 보안팀 알림
- [ ] EMP-003 사유 드롭다운 + 본인확인 사유 강제 + 매월 동의 재확인
- [ ] EMP-007 1천만원 이상 현금거래 CTR 자동 등록 (EMP-016 연계)
- [ ] EMP-008 정정 시 원거래+정정거래 둘 다 WORM 보존
- [ ] EMP-012 심사자 ≠ 실행자 직무분리 + KYC 재스크리닝
- [ ] EMP-013·014 연계: 91일+ → 부실 → 상각/매각
- [ ] EMP-015 법원 송달 즉시 등록 + 출금 차단
- [ ] EMP-018 11개 배치 + DR 동기화 모니터
- [ ] EMP-022 신청자 ≠ 승인자 직무분리 + 결재 라인 자동 산출
- [ ] EMP-023 약관 변경 14일 사전 통지 + 광고규제 검수
