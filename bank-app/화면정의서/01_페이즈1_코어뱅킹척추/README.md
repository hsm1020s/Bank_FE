# 페이즈 1 — 코어뱅킹 척추 (Core Banking Spine)

## 1. 페이즈 목표

고객이 **로그인 → 자산 조회 → 이체 → 대출 신청·상환**의 메인 도메인 흐름을 전 영역 끊김 없이 수행 가능한 상태로 만든다. 이 페이즈가 끝나면 사용자는 **현금이 흐르는 시스템**을 손에 쥐게 된다.

## 3. 페이즈 1 완료 정의 (Definition of Done)

- [ ] 9개 화면 전체 라우팅 + 권한 가드 구현
- [ ] 모킹 API 엔드포인트 13개 응답 (`00-08` 참조)
- [ ] 인증등급 1·2·3단계 승급 모달 동작
- [ ] FDS 룰 5개 차단·통과·지연 시나리오 동작
- [ ] 거래 발생 시 감사로그 표준 필드 적재 (콘솔 또는 로컬)
- [ ] 적합성·적정성 진단(CUS-007, CUS-013) 결과 화면 노출
- [ ] 핵심설명서 PDF 모킹 + 음성 안내 토글
- [ ] DSR 실시간 계산 + 40% 경고 / 50% 차단 (CUS-013)
- [ ] 가계대출 숙려기간 2영업일 카운트다운 (CUS-013)
- [ ] 거래내역 5년 + 이자지급 시 세금 분리 표시 (CUS-006)
- [ ] 일배치 BAT-001/002/003 모킹으로 일일 이자 발생 / 연체 단계 전이 동작
