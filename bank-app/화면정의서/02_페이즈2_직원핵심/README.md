# 페이즈 2 — 직원 핵심 (Employee Core)

## 1. 페이즈 목표

직원이 **KPI 모니터링 + 여신 심사 + 연체 관리**의 코어 직무를 수행 가능하도록 한다. 고객 채널의 신청·이체가 흘러 들어와 직원이 처리·결정·회수하는 백오피스 흐름.
